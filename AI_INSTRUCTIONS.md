# AI Instructions

This repo's shared working procedure is in:

- `docs/operating-procedure.md`

All AI assistants should read that file before making workflow, Git, deployment, or release decisions. Tool-specific entrypoints also point here:

- `AGENTS.md` for Codex/OpenAI-style agents
- `CLAUDE.md` for Claude
- `.github/copilot-instructions.md` for GitHub Copilot

If an AI tool does not automatically load these files, paste or reference `docs/operating-procedure.md` in the chat before asking it to make project changes.
